JavaScript

Porque aprender JavaScript

JavaScript é essencial para qualquer site web bacana. É a linguagem que traz vida para o site, possibilita conteúdo dinâmico, elementos interativos e design responsivo

  • Todo site interativo usa JavaScript para criar experiências envolventes para o usuário.
  • Funciona perfeitamente com HTML, CSS e frameworks modernos como React, Angular e Vue.
  • JavaScript roda em qualquer lugar: navegadores, servidores, aplicativos móveis,desktop e IoT.
  • JavaScript melhora o desempenho, a experiência do usuário e a funcionalidade do aplicativo.
  • De simples animações a aplicativos web complexos, o JavaScript fornece interatividade para tudo.

Camadas de Desenvolvimento

Com a chegada dos padrões Web,o conceito de desenvolvimento em camadas tornou-se um importante ponto a ser considerado na contrução de aplicações Web.

Tal conceito preconiza a separação dos códigos em 3 camadas

Conteúdo - HTML (Fornece a estrutura da página)

Formatação ou Estilo - CSS

Comportamento - JS (adiciona interatividade)

Como inicar JavaScript

Você pode escrecer código JavaScript diretamente no arquivo HTML com o elemento

< script > < /script >

Comentários

Comentários são pequenos textos dentro do código com a funcionalidade de facilitar o entendimento do código

//comentário em linha única

<!-- comentário em multiplas linhas

Caixas de Diálogo

São janelas pop-up que se destinam a fornecer informações ao usuário

Existem 3 tipo de janelas pop-up

  • Caixa de alerta
  • Caixa de confirmação
  • Caixa de entrada

Objeto Document do JavaScript

O objeto document em JavaScript é a ponte que permite aos scripts interagir com o conteúdo de uma página
Ele representa a estrutura do documento HTML como uma árvore, o Modelo de Objeto de Documento (DOM),
e oferece métodos para acessar e modificar essa estrutura

Escrever HTML com JavaScript

Um exemplo do uso do objeto Document é para escrever um texto na página HTML

document.write("código HTML")
